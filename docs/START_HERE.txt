🚀 FIRST-TIME SETUP

  1. Open Terminal/PowerShell/Command Prompt
  2. Navigate to this directory
  3. Run: python setup.py
  4. Wait for completion (takes ~2 minutes)
  5. Edit config/.env with your AWS credentials (if needed)
  6. Run: python experiments/01_verify_infrastructure.py

⚙️ CONFIGURATION

  1. Copy template: cp config/.env.template config/.env
  2. Edit: config/.env
  3. Add your AWS credentials:
     AWS_ACCESS_KEY_ID=your_key
     AWS_SECRET_ACCESS_KEY=your_secret
     AWS_DEFAULT_REGION=us-east-1

🆘 TROUBLESHOOTING

  Problem: "python: command not found"
  Solution: Use "python3" or add Python to PATH
  Help: See docs/CROSSPLATFORM_GUIDE.md

  Problem: "aws: command not found"
  Solution: Install AWS CLI v2
  Help: See docs/guides/PHASE4_5_QUICK_START.md

  Problem: "AWS credentials not found"
  Solution: Set environment variables or ~/.aws/credentials
  Help: See docs/guides/PHASE4_5_QUICK_START.md
